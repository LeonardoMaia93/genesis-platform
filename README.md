# Genesis Platform

> Plataforma para gerenciamento de eventos, caravanas e presença de igrejas.

## 📖 Sobre o Projeto

O Genesis Platform é uma plataforma desenvolvida para facilitar a organização de eventos religiosos, permitindo que membros confirmem presença em eventos, enquanto organizadores gerenciam inscrições, embarques, listas de presença e relatórios.

O sistema será composto por um aplicativo móvel para membros e um painel administrativo para organizadores.

---

## 🎯 Objetivos

- Gerenciar eventos da igreja.
- Controlar confirmações de presença.
- Gerenciar embarque e retorno.
- Utilizar QR Code para check-in.
- Disponibilizar painel administrativo.
- Gerar relatórios em PDF e Excel.
- Enviar notificações automáticas.
- Suportar múltiplas igrejas.
